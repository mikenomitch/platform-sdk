New Outbound Worker

About Outbound Workers

Outbound Workers intercept all fetch() requests made by tenant Workers. Use them to:

  • Enforce security policies
  • Add request logging
  • Route through proxies
  • Block unauthorized domains

New Tail Worker

About Tail Workers

Tail Workers receive logs and traces from tenant Workers after execution. Use them for:

  • Custom logging pipelines
  • Analytics and metrics
  • Error tracking
  • Compliance auditing
